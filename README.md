# A-Study-in-Python
python零基础学习，新手编程，小白友好，请配合Anaconda的Jupyter notebook使用。

此学习库主要针对编程小白，基于网络资源进行修改汇编，欢迎大家提出修改意见，通过探讨一起提高。
